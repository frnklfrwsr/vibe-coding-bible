# The Vibe Coding Bible

This site is a human-readable navigation layer for the Vibe Coding Bible.

The canonical source remains the Markdown repository on GitHub:

[frnklfrwsr/vibe-coding-bible](https://github.com/frnklfrwsr/vibe-coding-bible)

Use this site to find the right source file quickly. Use the GitHub repository for the authoritative text, history, issues, pull requests, and release artifacts.

## Canonical-source rule

The docs site does not replace the repository. It intentionally links to canonical Markdown files instead of duplicating the full VCB corpus into a second unmanaged tree.

If the site and repository ever disagree, treat the GitHub repository as authoritative.
